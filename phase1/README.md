# Todo Console App - Phase 1

Simple in-memory todo list manager built with spec-driven development.

## Features
- ✅ Add tasks with title and description
- ✅ View all tasks with status indicators
- ✅ Update task details
- ✅ Delete tasks with confirmation
- ✅ Mark tasks complete/incomplete
- ✅ Clean menu navigation

## Setup
Requires Python 3.13+

## Run
```bash
cd src
python main.py
```

## Note
Data is stored in memory only and resets when the program exits (as per Phase 1 requirements).

## Technologies
- Python 3.13+
- Standard library only
- Spec-driven development with Claude Code
